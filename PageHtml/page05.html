HTML
Language de description de pages utilisé pour le Web. Ce langage se compose de balises qui seront interprétées par le navigateur du poste client, de texte et d'images qui seront affichées.
Navigateur
Il s'agit d'un programme interprétant le code HTMLpour afficher une page Web. Les deux navigateurs les plus communément utilisées sont :
  • Mozilla FireFox
  • Internet Explorer
Web
Ensemble de sites mettant à la disposition des internautes une multitudes de ressources (textes,images, sons...). Chaque page du Web est munie de liens permettant la navigation.
